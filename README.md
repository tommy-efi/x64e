# x64e
### This is a lightweight x64 emulation library written in C++ that you can easily integrate into your own project. Designed for real-time pointer decryption. It allows "external" applications to emulate x64 assembly from another running application. It's really fast


- Fast and lightweight x64 instruction emulation
- Easy integration into existing projects


### Note: This does **NOT** support all x64 instructions yet. If you've run into an instruction that is not supported, implementation is easy enough. Look through [this](https://www.felixcloutier.com/x86/) to implement instructions you may need.
